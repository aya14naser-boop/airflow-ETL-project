airflow-ETL-project

Overview
This project demonstrates an end-to-end ETL pipeline using Apache Airflow to process sales data from a CSV file.

---

 Pipeline Stages

 1. Extract
- Read raw sales data from CSV

 2. Transform
- Clean column names
- Handle missing values
- Convert numeric columns
- Calculate profit

 3. Load
- Output cleaned data for analysis

---

 Tools
- Python
- Pandas
- Apache Airflow
